## Description:
The aim of this project is to create a small fan page for the Culling Games whilst also following all of the mandatory criteria for the project set by our university course. The Culling Games are a sort-of battle royale type of game where players earn points by "eliminating" other players. The idea is to have a section for the rules of the game, a section for viewing the contestants, their status, and current scores, then a section for registering yourself as a player and another section for voting for your favourite character as this is meant for fans of the series. The project will be a one-page website.
